## Analysis of the rhizospheres of dominant plant species in tropical plantations
Repository for the code to analyze the fungal and bacterial communities of the rhizosphere of _Acacia mangium, Coccos nucifera, Swietenia macrophylla, Pterocarpus indicus, Saccharum spp._ and _Imperata cylindrica_.

Information on the location of raw sequences is reported in Fungi and Bacteria folders, along with codes and metadata used.
